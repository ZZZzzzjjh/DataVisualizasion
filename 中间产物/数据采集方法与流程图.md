# 数据采集方法与流程图

```mermaid
flowchart TD
    A["确定研究主题<br/>城市就业与生活成本分析"] --> B["确定研究对象<br/>选取20个代表性城市"]
    B --> C["明确数据需求<br/>就业、经济人口、生活成本、地图边界"]

    C --> D1["网络招聘数据采集"]
    C --> D2["宏观经济统计数据采集"]
    C --> D3["生活成本数据采集"]
    C --> D4["地图边界数据采集"]

    D1 --> E1["人社部门公开资料<br/>Job-SDF数据集<br/>招聘平台公开报告"]
    D2 --> E2["国家统计局国家数据<br/>中国统计年鉴<br/>中国城市统计年鉴<br/>各城市统计公报"]
    D3 --> E3["Numbeo中国城市生活成本数据<br/>生活成本指数、租金指数、购买力指数"]
    D4 --> E4["阿里云DataV GeoAtlas<br/>GeoJSON地图边界数据"]

    E1 --> F["整理为结构化表格"]
    E2 --> F
    E3 --> F
    E4 --> F

    F --> G["字段统一<br/>城市名称、指标名称、统计年份"]
    G --> H["数据清洗<br/>去重、去脚注、处理异常值"]
    H --> I["缺失值处理<br/>保留并标记缺失字段"]
    I --> J["数据合并与关联<br/>以城市名称为主键合并"]
    J --> K["指标标准化与特征构建<br/>人均GDP、收入成本比、就业生活性价比指数"]
    K --> L["形成城市综合数据集<br/>用于统计分析、建模与可视化"]
```

## 报告正文引用示例

本文的数据采集流程如图 2-1 所示。首先确定研究主题和研究城市，然后根据就业、经济人口、生活成本和地图边界等分析需求分别采集数据。随后将不同来源的数据整理为结构化表格，并进行字段统一、数据清洗、缺失值处理和数据合并，最终形成城市综合数据集，为后续统计分析、建模和可视化展示提供数据基础。
