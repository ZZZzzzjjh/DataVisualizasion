# 3 城市就业与生活成本描述性统计分析与可视化

本章基于整理后的城市综合数据，对 20 个样本城市的宏观发展水平、就业市场结构、生活成本水平以及就业收入与生活成本的匹配关系进行分析。本文不是只单独比较某一个指标，而是把 GDP、人口、薪资、生活成本、房租和月结余等指标结合起来观察，从而更真实地反映毕业生选择就业城市时可能面临的机会和压力。

如图 3-1 所示，本章的分析框架从城市宏观背景、就业市场结构和生活成本水平三个方面展开，再进一步分析收入成本匹配关系，最后识别不同城市类型并形成城市选择参考。

<a id="fig3-1"></a>

![图3-1 描述性统计分析框架图](D:/DataVisualization/大作业/zjh/figures/图3-1_描述性统计分析框架图.png)

图 3-1 描述性统计分析框架图

从图 3-1 可以看出，本章并不是孤立比较某一个指标，而是先建立城市背景，再分析就业和成本，最后把收入与支出放在同一框架下判断城市是否适合就业生活。

## 3.1 城市宏观发展背景分析

城市宏观发展背景能够反映一个城市的经济基础、人口规模和区域特征。虽然宏观指标不能直接说明个人生活质量，但它可以为后续就业市场和生活成本分析提供基础背景。

### 3.1.1 GDP 规模与人均 GDP 对比

如图 3-2 所示，上海、北京、深圳、重庆和广州位于样本城市 GDP 规模前列，其中上海和北京的经济总量明显高于其他城市。这说明一线城市和部分直辖市在经济体量方面仍然具有较强优势。

<a id="fig3-2"></a>

![图3-2 GDP规模对比图](D:/DataVisualization/大作业/zjh/figures/图3-2_GDP规模对比图.png)

图 3-2 GDP 规模对比

从图 3-2 可以看出，GDP 规模较大的城市通常产业基础较强，就业资源也更加集中。不过，GDP 只能反映城市整体经济总量，不能直接说明普通毕业生在该城市就业后的实际生活压力。因此，还需要进一步结合人均 GDP、薪资和生活成本进行分析。

如图 3-3 所示，宁波、苏州、重庆、长沙和青岛在人均 GDP 指标上表现较突出。部分城市虽然总体 GDP 不是最高，但由于人口规模相对较小或产业效率较高，人均 GDP 表现反而较好。

<a id="fig3-3"></a>

![图3-3 人均GDP对比图](D:/DataVisualization/大作业/zjh/figures/图3-3_人均GDP对比图.png)

图 3-3 人均 GDP 对比

结合图 3-2 和图 3-3 可以发现，城市经济实力不能只看 GDP 总量。对于毕业生来说，经济规模大的城市可能机会更多，但人均 GDP 较高的城市也可能具备较好的产业基础和发展潜力。

### 3.1.2 人口规模与城市等级分布

人口规模可以反映城市吸引力和市场容量。如图 3-4 所示，上海、北京、深圳、广州和成都的人口规模较大，说明这些城市具有较强的人口集聚能力。人口规模较大通常意味着消费市场更大、就业机会更多，但也可能带来住房紧张和通勤压力。

<a id="fig3-4"></a>

![图3-4 人口规模对比图](D:/DataVisualization/大作业/zjh/figures/图3-4_人口规模对比图.png)

图 3-4 人口规模对比

从图 3-4 可以看出，一线城市和部分新一线城市人口吸引力较强。对于毕业生来说，人口规模大的城市机会相对更多，但竞争也可能更激烈，因此不能只根据人口规模判断城市是否适合就业。

如表 3-1 所示，本文样本既包括北京、上海、广州、深圳等一线城市，也包括杭州、南京、成都、武汉、长沙等新一线或重要二线城市。这样的样本设置可以较好地覆盖不同等级城市之间的差异。

表 3-1 样本城市等级分布

|---|---|---|

| 新一线及重要二线城市 | 杭州、南京、成都、武汉、西安、长沙等 | 兼具就业机会和相对可控的生活成本 |

结合图 3-4 和表 3-1 可以看出，本文样本既包含高能级城市，也包含生活成本相对较低的新一线和二线城市，这为后续比较“高薪高成本”和“低成本友好”城市提供了基础。

### 3.1.3 区域发展差异分析

从区域角度看，华东、华南、华北、西南、华中和西北等区域的城市在经济基础和产业结构上存在差异。如图 3-5 所示，不同区域城市在平均月结余方面表现不同。部分华东和西南城市在薪资与生活成本匹配方面较为均衡，而华北部分城市由于租金压力较高，实际月结余表现不一定占优。

<a id="fig3-5"></a>

![图3-5 区域平均月结余对比图](D:/DataVisualization/大作业/zjh/figures/图3-5_区域平均月结余对比图.png)

图 3-5 区域平均月结余对比

由图 3-5 可以看出，区域差异不仅体现在经济规模上，也体现在生活压力和实际可支配收入上。因此，后续分析不能只比较单个城市，还要关注城市所在区域和城市等级带来的综合影响。

就业市场结构是本文分析的重要内容。毕业生选择城市时，除了关注工资高低，还需要关注一个城市有哪些主要行业、岗位类型是否适合自己、学历和经验要求是否匹配。

## 3.3 城市生活成本水平分析

生活成本直接影响毕业生在城市中的实际生活压力。本文主要从单人月生活成本、市中心一居室租金以及租房压力三个角度进行分析。

### 3.3.1 单人月生活成本对比

如图 3-11 所示，样本城市单人月生活成本平均约为 3626 元/月，其中上海、苏州、广州、北京和深圳较高。上海约为 4228 元/月，苏州约为 4125 元/月，广州约为 3975 元/月，北京约为 3949 元/月，深圳约为 3946 元/月。

<a id="fig3-11"></a>

![图3-11 单人月生活成本对比图](D:/DataVisualization/大作业/zjh/figures/图3-11_单人月生活成本对比图.png)

图 3-11 单人月生活成本对比

从图 3-11 可以看出，经济发达城市的日常生活成本通常也较高。不过，不同城市之间日常生活成本的差距没有租金差距那么明显，因此还需要单独分析住房支出。

### 3.3.2 市中心一居室租金对比

如图 3-12 所示，样本城市市中心一居室租金平均约为 3168 元/月，但不同城市之间差距较大。北京约为 7238 元/月，上海约为 6363 元/月，深圳约为 5312 元/月，杭州约为 4172 元/月，广州约为 3644 元/月，均处于较高水平。

<a id="fig3-12"></a>

由图 3-12 可以看出，北京、上海、深圳等高薪城市同时也是租金压力较大的城市。相比之下，合肥、长沙、佛山、西安和天津的市中心一居室租金较低，住房压力相对较小。

为了综合观察生活成本与租房压力，本文将单人月生活成本和市中心一居室租金进行对比。如图 3-13 所示，部分城市同时具有较高的生活成本和较高的租金，例如北京、上海、深圳、杭州等；而长沙、成都、宁波、青岛等城市租金压力相对较小，整体生活压力更容易控制。

结合图 3-11、图 3-12 和图 3-13 可以发现，生活成本和租金共同决定了城市生活压力。对于毕业生来说，房租通常是每月最大的固定支出，因此即使一个城市工资较高，如果租金过高，实际生活压力仍然会比较大。

## 3.4 就业收入与生活成本匹配关系分析

本节将税后月薪、生活成本、租金和月结余结合起来分析，重点观察不同城市在收入和成本之间是否匹配。

### 3.4.1 税后月薪与生活成本关系分析

如图 3-14 所示，税后月薪和单人月生活成本之间存在一定正相关关系。上海、北京、深圳、杭州等城市税后月薪较高，但生活成本也较高。这说明高薪城市往往伴随着较高消费水平。

<a id="fig3-14"></a>

### 3.4.4 高薪高成本、低成本友好与均衡型城市识别

为了进一步总结不同城市的特征，本文根据薪资水平、生活成本、租金和月结余对城市进行初步分类。如图 3-17 所示，税后月薪与实际月结余四象限图可以帮助识别不同类型城市。

图 3-17 高薪高成本、低成本友好与均衡型城市识别

从图 3-17 可以看出，北京、上海、深圳、杭州等城市属于较典型的高薪高成本型城市，这类城市就业机会多、薪资水平高，但生活和住房压力也大。长沙、成都、南京、宁波、青岛等城市属于相对均衡或低成本友好型城市，虽然薪资不一定最高，但生活成本和租金较适中，实际月结余较好。对于预算敏感型毕业生来说，这类城市可能更适合长期生活。

综上，本章通过多张图表分析了城市宏观背景、就业结构、生活成本和收入成本匹配关系。结果表明，城市就业选择不能只看 GDP 和平均工资，还需要综合考虑租金、生活成本、岗位结构和实际月结余。对于毕业生来说，真正有参考价值的是收入和支出共同作用后的结果，而不是单一指标排名。
