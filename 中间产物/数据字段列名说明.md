# 数据字段列名说明

本项目在程序内部使用英文列名，在仪表盘和报告中展示中文列名。为了避免 Streamlit 运行时出现 `KeyError`，数据表建议保留以下字段。

| 英文字段名 | 中文列名 | 说明 |
| --- | --- | --- |
| `display_city` | 城市 | 页面展示用城市名 |
| `city_name_zh` | 城市中文名 | 城市中文名称 |
| `city_name_en` | 城市英文名 | 用于匹配 Numbeo 城市页面 |
| `province_zh` | 省份 | 城市所在省份或直辖市 |
| `region` | 区域 | 华东、华南、华北等区域 |
| `city_tier` | 城市等级 | 一线、新一线、二线等 |
| `gdp_2024_cny_million` | GDP（百万元） | 2024 年 GDP，单位为百万元 |
| `gdp_2024_cny_billion` | GDP（亿元） | 由程序换算得到，单位为亿元 |
| `population_2020` | 2020 年人口 | 第七次人口普查口径人口 |
| `population_2020_million` | 人口（百万人） | 由程序换算得到，单位为百万人 |
| `gdp_per_capita_2024_cny` | 人均 GDP（元） | 用于宏观背景对比 |
| `avg_salary` | 税后月薪（元/月） | Numbeo 城市页面税后月薪 |
| `monthly_cost_single_rmb` | 单人月生活成本（元/月，不含房租） | Numbeo 城市页面估算值 |
| `monthly_cost_family_rmb` | 四口之家月生活成本（元/月，不含房租） | Numbeo 城市页面估算值 |
| `rent_1br_city_centre_rmb` | 市中心一居室月租金（元/月） | Numbeo 城市页面租金数据 |
| `rent_1br_outside_centre_rmb` | 非中心一居室月租金（元/月） | Numbeo 城市页面租金数据 |
| `rent_3br_city_centre_rmb` | 市中心三居室月租金（元/月） | Numbeo 城市页面租金数据 |
| `rent_3br_outside_centre_rmb` | 非中心三居室月租金（元/月） | Numbeo 城市页面租金数据 |
| `monthly_balance_after_cost_rmb` | 扣除生活成本后月结余（元/月） | 税后月薪减单人月生活成本 |
| `monthly_balance_after_rent_cost_rmb` | 扣除生活成本和房租后月结余（元/月） | 税后月薪减生活成本和市中心一居室租金 |
| `salary_cost_ratio` | 月薪/单人生活成本 | 辅助判断收入覆盖生活成本能力 |
| `salary_rent_ratio` | 月薪/市中心一居室租金 | 辅助判断住房压力 |
| `job_market_proxy` | 招聘热度代理 | 由 GDP 和人口构造，仅用于辅助比较 |
| `city_type` | 城市类型 | 根据月薪、成本和结余自动划分 |
| `rmb_data_note` | RMB 数据说明 | 说明直接提取和估算补齐情况 |

## 就业结构补充表

文件位置：`data/cleaned/city_job_structure.csv`

| 英文字段名 | 中文列名 | 说明 |
| --- | --- | --- |
| `city_name_zh` | 城市 | 城市中文名称 |
| `region` | 区域 | 城市所属区域 |
| `city_tier` | 城市等级 | 一线、新一线、二线等 |
| `industry` | 行业类别 | 如互联网/软件、金融/商务服务、先进制造等 |
| `position_type` | 岗位/工种 | 如数据分析、产品运营、供应链运营、质量工程师等 |
| `job_share_percent` | 岗位占比（%） | 用于展示就业市场结构 |
| `position_avg_salary_rmb` | 岗位平均薪资（元/月） | 用于比较不同岗位薪资水平 |
| `education_requirement` | 学历要求 | 本科、大专及以上、硕士优先等 |
| `experience_requirement` | 经验要求 | 不限、1-3 年等 |
| `job_structure_source` | 就业结构数据说明 | 说明该表为课程项目就业结构补充表 |

## 最低必需列

如果重新整理数据，建议至少保留：

```text
city_name_zh
city_name_en
province_zh
region
city_tier
gdp_2024_cny_million
population_2020
avg_salary
monthly_cost_single_rmb
rent_1br_city_centre_rmb
```

其中 `display_city`、`gdp_2024_cny_billion`、`population_2020_million`、`monthly_balance_after_cost_rmb`、`monthly_balance_after_rent_cost_rmb` 和 `city_type` 会由程序自动补齐。
